# 🧠 CNN Image Classification

A professional end-to-end image classification platform built with TensorFlow and Streamlit. Create datasets, train CNN models, evaluate performance, and make real-time predictions.

## ✨ Features

- **Dataset Manager** – Create classes, upload images (30+ per class), webcam capture, real-time statistics
- **CNN Training** – 4-layer CNN with BatchNorm, Dropout, live training metrics with Plotly
- **Evaluation** – Confusion matrices, classification reports, per-class performance analysis
- **Real-time Prediction** – Upload/capture images, instant classification with confidence scores
- **Modern UI** – Professional design with light/dark theme, responsive layout
